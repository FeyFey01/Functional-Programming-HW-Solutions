# BLG 458E - Functional Programming (Haskell) Coursework

This repository contains the homework assignments and the term project for the **BLG 458E Functional Programming** course at Istanbul Technical University.  
The coursework focuses on implementing mathematical models and simulations using **functional programming principles** in **Haskell**.

---

## Programming Language & Environment

- **Primary Language**: [Haskell](https://www.haskell.org/)
- **Supporting Tools**:
  - **Julia**: Used for generating grid maps and initial simulation parameters
  - **CloudCompare**: Used for 3D visualization of generated STL files
- **Compiler**: GHC (Glasgow Haskell Compiler)

---

## Coursework Overview

### [Homework 1: Hilbert Curves](https://github.com/FeyFey01/Functional-Programming-HW-Solutions/tree/main/hw1)
- **Task**: Implement a recursive algorithm to generate **Hilbert Curves**
- **Demonstrates**:
  - Recursive decomposition
  - Functional abstraction
  - 3D output generation via STL files

---

### [Homework 2: Schelling Segregation Model](https://github.com/FeyFey01/Functional-Programming-HW-Solutions/tree/main/hw2)
- **Task**: Simulate social segregation using the **Schelling agent-based model**
- **Demonstrates**:
  - Functional processing of 2D grid data
  - Rule-based agent movement
  - Tolerance-based decision logic

---

### [Homework 3: Shortest Distance in Knowledge Graphs](https://github.com/FeyFey01/Functional-Programming-HW-Solutions/tree/main/hw3)
- **Task**: Compute the shortest path between two entities in a large knowledge graph
- **Demonstrates**:
  - Graph traversal algorithms
  - Efficient handling of large datasets
  - Path reconstruction and error handling

---
